MyStudio - Start by WGSN

A seamless SaaS experience for fashion professionals.

Overview

Start by WGSN is a SaaS platform designed for fashion stylists, providing essential image-based insights that inform, inspire, and support decision-making and collaboration.

The aim of this project was to enhance user experience and usability through UI design improvements and usability testing, ensuring a seamless workflow for fashion professionals.

My role

Product Designer | Dicas UX

User Research, interaction, UI design, prototyping & testing

Problem

Fashion professionals relied on fragmented tools for trend research and collaboration, making it difficult to find relevant insights and streamline decision-making. The platform needed a more intuitive UI and improved usability to enhance user engagement and workflow efficiency. My goal was to create a seamless, image-driven experience that simplified information access and encouraged collaboration.

KPI's

  • User engagement rate
  • Time spent per session
  • Task completion efficiency
  • User retention rate
