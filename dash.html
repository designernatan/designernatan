The brief

Overview

The Analytical B2B Dashboard was developed for managers in corporate learning environments. It provides real-time insights into employee engagement, course completion rates, and licensing status, allowing companies to make data-driven decisions.

My role

Product Designer

User Research, UI/UX Design, Prototyping, and Usability Testing

The challenge

Problem & objectives

Problem

Before the redesign, managers had limited visibility into employee learning activities. The previous interface was a simple chart with a table below, lacking actionable insights. The goal was to create a structured, insightful dashboard that could generate reports and facilitate data-driven decision-making.

KPI's

  • — Churn Rate Reduction
  • — Increase in Sales
  • — Growth in Upsell Transactions
  • — Improved User Engagement
Methodology

Process

Research Usability Test UI Improvements Delivery

Using an adaptive Double Diamond approach, I conducted user interviews to understand the main pain points. The research revealed issues with data visibility, lack of structure, and difficulty generating meaningful reports.

Based on these insights, the dashboard was redesigned with clear visual hierarchies, interactive elements, and better contrast. Iterative usability tests ensured an optimal user experience.

Design solution

Final product

The final dashboard provided a structured UI with actionable insights, allowing managers to monitor employee engagement and licensing with ease. Features like an intuitive walkthrough, improved chart contrast, and hover interactions made navigation smoother and data more digestible.
