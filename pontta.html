Overview

Pontta Gestor is a business management tool tailored for furniture retailers, including custom, modular, and mixed-format stores. The goal was to modernize a legacy system with poor usability and outdated visuals.

My Role

Product Designer

UI Design, UX Review, Usability Testing, Design System Implementation

Problem

The previous version of the system had a poor user experience and limited visual hierarchy. An initial wireframe provided by another team served as the starting point. The project required adapting the interface to Googleâ€™s Material Design guidelines while enhancing usability for retail professionals.

KPI's

  • Task efficiency improvement
  • Reduced onboarding Time
  • User error rate reduction

Process

Review
Wireframe
UI Design
Test

Starting with a rough wireframe from another team, I mapped usability flaws and restructured the layout based on Material Design principles. The visual interface was completely redesigned to offer better clarity, consistency, and modern aesthetics.

I conducted an in-store usability test at one of Pontta's furniture shops, validating flows with a real user in a real context. Feedback from this session led to minor UI refinements before final delivery.

Design Solution

Final Product

The final interface adhered to Material Design guidelines and introduced meaningful hierarchy, responsive components, and better data presentation. The result was a tool that felt modern and usable, even for employees unfamiliar with digital systems.
